#project

Project Sentinel is dedicated to designing an autonomous drone swarm, intended for environmental monitoring and reforestation efforts. it leads this project. It operates with a budget of $121k, utilizing it as its primary operational tool. The aim is to leverage drone technology for ecological restoration and observation.

## Key facts
- Goal: Design an autonomous drone swarm for environmental monitoring and reforestation.
- Lead: [[Chloe Zhao]]
- Run by [[Chronos Systems]]
- Built with [[AetherNet Monitor]]
- Budget: $121k
