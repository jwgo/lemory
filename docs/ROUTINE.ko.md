# 🍋 세컨드브레인 루틴

핵심은 하나예요. **캡처는 아무렇게나, 정리는 도구가, 검색은 대화로.**

## 처음 한 번만 (10분)

```bash
lemory up ~/Obsidian/MyVault              # 색인 + 대시보드
claude mcp add lemory -- lemory mcp --vault ~/Obsidian/MyVault --client claude-code
lemory skill install claude-code          # Claude가 알아서 검색부터 하게
lemory hooks install claude-code          # 세션 끝나면 자동으로 기억 저장
```

민감한 노트(일기, 인사 정보)에는 맨 위에 `lemory: false` 한 줄만 적으세요.
그 노트는 색인도, 어떤 모델로도 전송되지 않아요.

## 매일: 사실상 0분 (습관만)

- **캡처는 평소처럼 하세요.** 옵시디언 데일리노트든, 아무 편집기든요.
  Lemory를 거칠 필요가 없어요. 워처가 알아서 색인해요.
- **찾을 땐 검색하지 말고 물어보세요.** Claude에서 "지난주 회의에서 예산
  얼마로 정했지?"라고 하면 스킬이 검색부터 해요. 터미널이면
  `lemory ask`. 오타를 내도, 반말로 물어도 돼요.
- **결정이 나오면 그 자리에서** "이거 기억해줘" 한 마디면 돼요. 저장하면서
  기존 기억과 겹치면 바로 알려줘요. 잊어버려도 세션이 끝날 때 훅이 요약을
  남겨요.
- **쓰는 요령 두 가지** — 노트 제목은 사람·프로젝트·개념 같은 이름으로,
  본문엔 [[링크]]를 아끼지 마세요. 그 링크가 멀티홉 질문("아틀라스 리드가
  좋아하는 DB는?")의 연료예요.

## 매주: 5분 정원 가꾸기

```bash
lemory suggest-links      # 언급만 하고 링크 안 건 것들, 진짜만 [[링크]]로
lemory drift              # 깨진 링크, 방치된 중복 점검
lemory drift --prompt     # 결과를 에이전트에 붙여넣으면 알아서 고쳐요
```

대시보드(`lemory serve`) 피드도 한 번 훑어보세요. AI가 이번 주에 뭘
적었는지 보이고, 이상한 건 되돌리기 버튼 하나예요. 링크를 걸수록 그래프가
촘촘해지고, 그래프가 촘촘할수록 검색이 좋아져요. 이 선순환이 이 도구의
전부예요.

## 매달: 10분 조감

- `lemory graph --open`으로 볼트 전체를 보세요. 연결이 하나도 없는 노트는
  링크를 걸거나 정리하고, 뜻밖에 커진 허브는 그 주제가 자라고 있다는
  신호예요.
- `lemory import-chats conversations.json`으로 ChatGPT/Claude 대화를
  넣어두면 대화 속 결정들도 검색에 들어와요.

## 질문 요령

- 시간 표현은 그대로 쓰세요. "요새", "지난주", "3월에" — 알아듣고
  반영해요.
- 좁힐 땐 연산자를 쓰세요. `lemory search "tag:회의록 folder:2026 예산"`.
- 답이 미심쩍으면 인용 [n]의 원래 노트를 열어보세요. 근거가 없으면
  "모른다"고 답하게 만들어져 있어요.
